# 🍺 Brewery Inc | Downriver Minecraft Server

Brewery Inc is the premier organization on the **Downriver** server, dedicated to large-scale infrastructure, automated resource farming, and providing the highest quality gear (and "beer") to its members.

This repository contains the source code for our advanced, glassmorphism-style organization website.

---

## 🌐 Live Features

* **Real-Time Status:** Dynamically fetches the current player count from the Downriver server (`66.59.210.161`).
* **Persistent Navigation:** Uses cookie-based state management to remember which tab you were last viewing.
* **Seamless Integration:** A single-page architecture (SPA) that uses background blurring and CSS animations for a modern feel.
* **Member Directory:** Automatically pulls high-resolution Minecraft skins for the entire roster using the Minotar API.

## 🚀 Installation & Deployment

1.  Clone this repository or download the `index.html` file.
2.  Open `index.html` in any modern web browser to view locally.
3.  To deploy, simply upload the `index.html` file to **GitHub Pages**, **Netlify**, or **Vercel**.

*Note: The background image is hosted via Catbox. If the link expires, replace the `background-image` URL in the `<style>` section of the HTML.*
